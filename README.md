# mp-weixin-region-data
微信小程序省市区选择器数据字典

## 说明
数据来源微信小程序的省市区选择器，主要有两个用处：

- 自定义省市区选择器组件
- 服务端数据字典与客户端一致

## 数据格式
有两种模式

**multi**

省、市、区（县）的数据独立存放，通过 value 作为 key 进行关联。
```json
// province.json 省
{
    "value": "110000",
    "text": "北京"
}
// city.json 市
{
    "110100": [{
        "value": "110100",
        "text": "北京市"
    }]
}
// district.json 区
{
    "110100": [{
        "value": "110101",
        "text": "东城区"
    }, ...]
}
```
